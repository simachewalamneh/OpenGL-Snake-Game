# OpenGL Snake Game

This is a Snake Game developed using C++ with OpenGL and GLUT.

## Features
- Grid-based snake movement
- Apple food system
- Score tracking
- Game over detection
- Pause and restart
- Increasing speed

## Controls
Arrow Keys - Move snake
P - Pause game
R - Restart game

## Requirements
- OpenGL
- GLUT / FreeGLUT
- C++ Compiler
